This file describes steps to build mico web site:

1. install ant
2. copy ant-contrib-1.0b3.jar to ant_dir/lib/
3. ensure the folder of mico web site are named as "mico";
4. ensure the folders of mico web site and this file in are in the same folder; as:
   your_folder/build/readme.txt
   your_folder/mico/website_files
5. ensure "ant" command are in PATH;
6. run build.bat or build.sh(make sure the execution privilage).
